# Kangxi Radicals Unicode

Representation of the [Kangxi radicals](https://en.wikipedia.org/wiki/Kangxi_radicals) in Unicode.

The Kangxi radicals are represented by two code blocks:

1. [Kangxi Radicals](https://en.wikipedia.org/wiki/Kangxi_Radicals_(Unicode_block)) (2F00-2FDF, 214 code points)
1. [CJK Radicals Supplement](https://en.wikipedia.org/wiki/CJK_Radicals_Supplement) (2E80-2EFF, 115 code points)

These two code blocks are listed below.

## Kangxi Radicals

- Name: Kangxi Radicals
- Range: 2F00–2FDF
- Code points: 214
- Sources:
  - Unicode: <https://unicode.org/charts/nameslist/n_2F00.html>
  - Unicode (PDF): <https://unicode.org/charts/PDF/U2F00.pdf>
  - Wikipedia: <https://en.wikipedia.org/wiki/Kangxi_Radicals_(Unicode_block)>

The Kangxi Radicals code block includes the parent versions of all the Kangxi radicals, i.e. the traditional main versions of the radicals without their variants (if any). The variants of any of these radicals are included in the [CJK Radicals Supplement](#cjk-radicals-supplement) code block.

The ordering and numbering is fixed and corresponds to the traditional ordering of the Kangxi radicals (i.e., for example "Radical 11" deterministially means ⼊ (Enter) with code point U+2F0A).
