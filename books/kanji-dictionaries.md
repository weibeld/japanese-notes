## The Kanji Dictionary: Find Any Compound Using Any of Its Component Characters

| Property            | Value               |
|---------------------|---------------------|
| Author              | Wolfgang Hadamitzky (author) & Mark Spahn (translation) |
| Edition             | 1                   |
| Year of publication | 1996                |
| Publisher           | Tuttle              |
| Pages               | 1786                |
| Dimensions          | 16.51×23.5 cm       |
| Kanji coverage      | ~6000               |
| Link                | [Amazon US](https://a.co/d/6ZICfQY), [Amazon DE](https://amzn.eu/d/fEqpa8I) |

Notes: 

- Entries organised by the 79 radicals introducted by Hadamitzky
- Readings in romaji (no kana)
- Includes many compounds for each kanji
- Compounds organised by the position of the kanji in the compound
- Every compound is listed under the entries of all its constituent kanjis
- Very comfortable lookup thanks to coloured page tabs marking radical stroke count and kanji overview at the beginning of each radical section
- Does not include stroke order
- Cons:
  - Doesn't inlcude a radical index (only a reading index)
  - Does not inlude radical names anywhere
  - Limited set of additional information for each kanji (in contrast to [The Kodansha Kanji Dictionary](#the-kodansha-kanji-dictionary))

Conclusion:

- Good for seeing, discovering, and comparing compounds
- Not much additional information per kanji → resort to [The Kodansha Kanji Dictionary](#the-kodansha-kanji-dictionary)

## The Kodansha Kanji Dictionary

| Property            | Value               |
|---------------------|---------------------|
| Author              | Jack Halpern        |
| Edition             | 2                   |
| Year of publication | 2013                |
| Publisher           | Kodansha            |
| Pages               | 2112                |
| Dimensions          | 15.52×22.28 cm      |
| Kanji coverage      | ~5500               |
| Link                | [Amazon US](https://a.co/d/iJHnMyd), [Amazon DE](https://amzn.eu/d/2yTbIYD) |

Notes:

- Very comprehensive dictionary including a lot of data often not found in other dictionaries (e.g. Jōyō or non-Jōyō including grades, Unicode code point, frequency, Chinese form and reading, synonyms)
- Entries are ordered by SKIP method (i.e. left-right, top-bottom, enclosure, and block patterns plus stroke count) instead of radicals
- Includes radical index which lists all kanjis under their radical
- Detailed naming for radicals (including separate names for all variants)
- Readings only in kana (no romaji)
- Entries of compounds structured by the various meaning variants of the kanji
  - Compounds with the kanji at different positions are mixed in the same section (in contrast to, for example, [The Kanji Dictionary](#the-kanji-dictionary-find-any-compound-using-any-of-its-component-characters))
- Fewer compounds per kanji than, for example, [The Kanji Dictionary](#the-kanji-dictionary-find-any-compound-using-any-of-its-component-characters)

Conclusion:

- Very comprehensive, well-organised, and well-formatted dictionary
- Less compounds than [The Kanji Dictionary](#the-kanji-dictionary-find-any-compound-using-any-of-its-component-characters), however, more focus on the nuances of the meanings of the kanji
- For an overview of meanings this dictionary is better than [The Kanji Dictionary](#the-kanji-dictionary-find-any-compound-using-any-of-its-component-characters), however, for discovering and comparing compounds, the latter is better

## Kodansha's Essential Kanji Dictionary

| Property            | Value               |
|---------------------|---------------------|
| Author              | Kodansha            |
| Edition             | 1                   |
| Year of publication | 2012 (2002)         |
| Publisher           | Kodansha            |
| Pages               | 928                 |
| Dimensions          | 12.95×18.03 cm      |
| Kanji coverage      | ~2000               |
| Link                | [Amazon](https://a.co/d/7PZrJYl) |

- Uses the Kangxi radical indexing system
  - Strange set of radicals (TODO: investigate)
- Includes stroke order
- Readings only in kana (no romaji)
- Includes radical index and readings index
- Pros:
  - Lists on-yomi and kun-yomi compounds separately, which is a nice contrast to other dictionaries
  - Very readable layout
Cons:
  - Includes only compounds where the kanji is in the first position
  - On-yomi and kun-yomi are not listed together next to the head character like in most other dictionaries (but only contained in the compounds)
  - No concise translation for each kanji next to the head character like in many other dictionaries
  - Includes radical names only at the beginning of each radical section, but not in the radical index (not possible to easily compare with the _real_ set of 214 Kangxi radicals)

## The New Nelson Japanese-English Character Dictionary

| Property            | Value               |
|---------------------|---------------------|
| Author              | Andrew N. Nelson, John H. Haig |
| Edition             | 1                   |
| Year of publication | 1997                |
| Publisher           | Tuttle              |
| Pages               | 1616                |
| Dimensions          | 15.24×22.86 cm      |
| Kanji coverage      | ~7000               |
| Link                | [Amazon US](https://a.co/d/0PMfvnq), [Amazon DE](https://amzn.eu/d/3S4aJhn) |

Notes:

- Successor of [The Modern Reader's Japanese-English Character Dictionary](the-modern-readers-japaneseenglish-character-dictionary) (includes an additional set of 1200 characters)
  - Known as the "New Nelson" in contrast to its predecessor which is referred to just as "Nelson" or the "Classic Nelson"
  - Uses the traditonal Kangxi radical system in contrast to its predecessor, which uses the Radical Priority System
- Extensive translations of each kanji, covering verbs formed with this kanji
- Includes the Japan Industrial Standard (JIS) code and the index in Morohashi's Dai Kanwa Jiten for each character (both not really necessary)
- Includes an indication whether the character is a kyōiku kanji, other jōyō kanji, or a non-jōyō kanji
- Includes radical names, including separate names for variants
- Said to include more rare and traditional (e.g. literary) kanjis than other dictionaries
- Pros:
  - Universal Radical Index: lists each kanji under all the graphemes that one might suspect to be its radical
- Cons:
  - Lengthy translations of readings may make it hard to parse the readings, if there are many readings

| Property            | Value               |
|---------------------|---------------------|

| Year of publication | 1962 (2004)         |
| Publisher           | Tuttle              |
| Pages               | 1112                |

| Link                | [Amazon US](https://a.co/d/0EbaGuf), [Amazon DE](https://amzn.eu/d/4RUS1MN) |

- Known as the "Nelson" or "Classic Nelson" in contrast to [The New Nelson Japanese-English Character Dictionary](#the-new-nelson-japanese-english-character-dictionary) which is known as the "New Nelson"
- Still in print as _"The Original Modern Reader's Japanese-English Character Dictionary: Classic Edition"_
- Uses the Radical Priority System (introduced by Nelson)
  - Extension of the traditional Kangxi radical system
  - Uses the same 214 radicals as the traditional Kangxi system (TODO: verify)
  - About 12% of the kanjis have a different radical than in the traditional system
  - Claims that this system allows identifying radicals more easily
- Many compounds per kanji
- Compound section is not subdivided, but one big list ordered by the stroke count of the remaining characters
- Does not include stroke order
- Readings in romaji (no kana)
Cons:
  - Compounds do not include the kanji itself
  - Includes only compounds where the kanji in question is at the first position
